# FixItNow
Spot it. Report it. Fix it.
FixItNow is a real-time civic issue reporting platform that empowers citizens to quickly report problems like potholes, broken streetlights, water leaks, and garbage overflows using just a photo, a short description, and GPS location. Issues are automatically categorized and routed to the relevant city department for faster resolution.
